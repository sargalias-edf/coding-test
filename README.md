# Javascript Code Test

`BookSearchApiClient` is a simple class that makes a call to a http API to retrieve a list of books and return them.

You need to refactor the `BookSearchApiClient` class, and demonstrate in `example-client.js` how it would be used. Refactor to what you consider to be production ready code. You can change it in anyway you would like and should use typescript.

This exercise focuses primarily on the code. You’re not expected to include infrastructure or CI/CD, but you’re welcome to if you feel it adds value.

Please don't submit a pull request. Either send a zip file containing your solution (preferred) or create a github repository and share the link.
